# GDD Scheduler
[gdd-scheduler.surge.sh](http://gdd-scheduler.surge.sh/)

Lost keeping track of the sessions to attend in Google Developers Day (2017,India)?.This app helps you plan your sessions to attend at GDD. You can select and save the sessions you want to attend and generate a pdf.
